백 엔드란?

웹 브라우저가 주고받는 데이터를 기록하고 가져오는 등의 뒷 단의 일을 처리하는 기술들

백 엔드 개발자

백엔드에서 사용하는 언어

백엔드는 특성상 프로그래밍 언어의 선택이 자유롭고 다양하다. 대표적으로는 Java, JSP, Spring, MySQL이 있다.

Java

자바는 웹 개발을 한다면 빼놓을 수 없는 언어이다. 자바에 대해서는 따로 자세히 포스팅을 할 예정이다.

JSP

웹 페이지의 보여지는 부분을 서버와 연결해주고, 사용자에게 받은 데이터를 서버에 보내고, 그에 대한 서버의 응답을 받아 사용자에게 전송해주는 역할을 한다. 같은 역할을 하는 언어로 PHP, ASP 그리고 Node.js가 있다.

Spring

Java 플랫폼을 위한 프레임워크로, 동적인 웹 사이트를 개발하기 위한 여러가지 서비스를 제공한다. 대한민국 공공기관의 웹 서비스 개발 시 사용을 권장하고, 전자정부 표준프레임워크로 쓰이고 있다.

MySQL

데이터베이스에서 데이터의 CRUD를 수행하는 언어이다.
